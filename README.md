# Hack4Us Website

This is the repo for the [Hack4Us](https://hack4us.ca) website.
It's built with Next.js and Convex. 

This website features a fully-fledged attendee/volunteer application system, with a review and ranking system for the organizers.

## Features
- Google & GitHub OAuth with better-auth
- User profile system with roles (attendee, volunteer, organizer)
- Attendee/volunteer application system
- Review and ranking system for the organizers
- Admin dashboard for the organizers
- AI-powered pre-scoring for the applications
- Z-score normalization for reviewer scores

## Tech Stack
- [Next.js](https://nextjs.org/)
- [Convex](https://www.convex.dev/)
- [better-auth](https://better-auth.com/)
- [Vercel AI SDK](https://sdk.vercel.ai/)
- [OpenRouter](https://openrouter.ai/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Shadcn UI](https://ui.shadcn.com/)
- [Lucide Icons](https://lucide.dev/)
